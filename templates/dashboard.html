How to Start the Stream

  1. Open BUTT (Broadcast Using This Tool).
  2. Configure BUTT to connect to your Icecast server:
    • Server Address: localhost
    • Port: 8000
    • Password: your-source-password
    • Mount Point: /stream
  3. Click the Start button in BUTT to begin streaming.
